=====================================
 PDF RESOURCES FOLDER
=====================================

Place your PDF files here!

INSTRUCTIONS:
-------------

1. Copy your PDF files into this folder

2. Rename files with clear names (use lowercase, no spaces):
   - Good: ro-purifier-brochure.pdf
   - Bad: RO Purifier Brochure.pdf

3. Update js/data.js with the file path

EXAMPLE:
--------

If you add a file: "company-profile.pdf"

Then in js/data.js, add:

{
    id: 22,
    title: "Company Profile 2026",
    icon: "fa-briefcase",
    fileType: "PDF",
    fileSize: "3 MB",
    url: "pdfs/company-profile.pdf",
    category: "Brochures"
}
